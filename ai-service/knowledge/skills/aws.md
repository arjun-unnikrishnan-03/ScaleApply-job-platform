# AWS Knowledge Base

Amazon Web Services (AWS) is a comprehensive, evolving cloud computing platform provided by Amazon that includes a mixture of infrastructure as a service (IaaS), platform as a service (PaaS) and packaged software as a service (SaaS) offerings.

## Core Services
- **EC2 (Elastic Compute Cloud)**: Resizable compute capacity in the cloud.
- **S3 (Simple Storage Service)**: Object storage service offering industry-leading scalability, data availability, security, and performance.
- **RDS (Relational Database Service)**: Easy to set up, operate, and scale a relational database in the cloud.
- **Lambda**: Serverless compute service that lets you run code without provisioning or managing servers.

## Architecture Principles
- Design for failure.
- Decouple components.
- Implement elasticity.
- Think parallel.
