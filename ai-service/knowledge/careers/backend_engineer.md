# Backend Engineer Career Progression

## Level 1: Junior Engineer (L1)
- **Focus**: Learning, executing well-defined tasks, and understanding the codebase.
- **Skills**: Basic language proficiency, writing unit tests, basic Git usage.
- **Scope**: Single components or bug fixes.

## Level 2: Mid-Level Engineer (L2)
- **Focus**: Independent execution, system design, and mentoring juniors.
- **Skills**: Advanced language features, API design, database optimization, CI/CD understanding.
- **Scope**: Entire features or microservices.

## Level 3: Senior Engineer (L3)
- **Focus**: Architecture, cross-team technical leadership, and resolving ambiguity.
- **Skills**: Distributed systems design, performance profiling, advanced DevOps, observability.
- **Scope**: Entire sub-systems, major architectural shifts.

## Level 4: Staff Engineer (L4)
- **Focus**: Organizational impact, setting technical vision, and strategic planning.
- **Skills**: Technical strategy, deep domain expertise, high-level system architecture.
- **Scope**: Department-wide technical direction, cross-functional initiatives.
