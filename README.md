# syncup-job-platform

A scalable job matching platform with AI integration built as a Full Stack Developer Assignment.

## Features
1. User Authentication (JWT)
2. Job Posting & Application
3. Resume Upload to AWS S3
4. AI-based Resume Matching using Azure OpenAI
5. Real-time Notifications via WebSockets
6. Redis caching for job search

## Tech Stack
- Node.js & Express
- Next.js 16 (App Router)
- MongoDB
- Redis
- AWS S3
- Socket.IO
- Tailwind CSS v4
